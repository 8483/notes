# Virtualbox

1. Install VirtualBox.
2. Download OS ISO.
3. Create VM.
4. Install OS.
5. Unmount ISO.
6. Download Guest Additions via VirtualBox.
7. Mount Guest Additions via VirtualBox.
8. Install Guest Additions via the OS.
9. **Reboot.**
10. Add clipboard settings.
11. Add host shared folder. It is under `/media/user/sf_folder_name`.
12. `sudo adduser username vboxsf` to access shared folder.
13. **Reboot.**
